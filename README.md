# Read Me du projet 2604 Application Tag reader
## But du projet
Le but de ce projet est de porter le logiciel du TP d’ELCO de RFID pour la lecture et écriture de carte RFID.
Le développement consistera à analyser le comportement de l'application existante sur machine virtuelle et à vérifier certains points avec le code originel en Visual Basic.
Pour en extraire les spécifications fonctionnelles et de protocoles et les implémenter intégralement en C# .NET pour de l’utilisation sur du Windows 10 voire 11.
